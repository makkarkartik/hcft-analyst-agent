HCFT RAG Agent — Anatomy in one picture

One request falls down the blue spine (control flow). Across each step sit the three concerns, always in the same order: 🔒 security guardrail → ↩ quality gate → 📊 offline twin. The dotted ⇄ links are the thesis: every live gate has an offline metric that guards it from regressing. Empty — cells and N/A badges are deliberate, not missing.
